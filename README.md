# จับผิดภาพ - Spot the Difference Game

ยินดีต้อนรับสู่เกม **จับผิดภาพ - Spot the Difference Game** เกมเว็บเบราว์เซอร์แนวหาจุดต่างระหว่างภาพถ่าย/รูปวาดประวัติศาสตร์ ที่มาพร้อมการออกแบบสไตล์ Glassmorphism ที่สวยงาม ทันสมัย และตอบสนองได้ดี

---

## 🎮 รายละเอียดของด่านในเกม (Game Description)
เกมนี้มีทั้งหมด 2 ด่าน ซึ่งประกอบด้วย:
1. **ด่านที่ 1: ยุทธหัตถี (ศึกช้างศึก)**
   - มีจุดต่างทั้งหมด **7 จุด**
2. **ด่านที่ 2: วิถีชีวิตกลางแม่น้ำป่า (Forest Boats)**
   - มีจุดต่างทั้งหมด **6 จุด**

---

## 🕹️ วิธีการเล่นและคำแนะนำ (Gameplay Instructions)

### 1. วิธีเปิดเกมเพื่อเล่น (How to Run)
คุณสามารถเข้าเล่นเกมได้โดยใช้วิธีใดวิธีหนึ่งดังต่อไปนี้:
* **เปิดโดยตรงผ่านเว็บบราวเซอร์:** ดับเบิลคลิกที่ไฟล์ [index.html](file:///Users/mac/spot-differences/index.html) ในโฟลเดอร์โครงการเพื่อเปิดเล่นได้ทันที
* **รันผ่านเว็บเซิร์ฟเวอร์จำลอง (Local Server):** สามารถติดตั้งและใช้เครื่องมือเช่น `Live Server` ใน VS Code หรือสั่งงานผ่านคำสั่ง Python/Node.js เพื่อเปิดเซิร์ฟเวอร์จำลองก็ได้ (เช่น `python3 -m http.server 8000` แล้วเข้าชมผ่าน `http://localhost:8000`)

### 2. วิธีเล่นเกม (How to Play)
* หน้าจอจะแสดงภาพ 2 ฝั่ง: ฝั่งซ้ายคือ **ภาพต้นฉบับ** และฝั่งขวาคือ **ภาพจับผิดจุดต่าง**
* ให้ค้นหาจุดที่เนื้อหาหายไปหรือแตกต่างจากภาพต้นฉบับทางขวา แล้ว**คลิกที่ภาพฝั่งขวา** ในบริเวณจุดต่างนั้น
* หากคลิก**ถูกจุด**: จะแสดงวงกลมสีแดงวงรอบจุดต่างที่พบ คะแนนจะเพิ่มขึ้น
* หากคลิก**ผิดจุด**: จะขึ้นสัญลักษณ์กากบาทสีแดงชั่วคราวและมีเอฟเฟกต์การสั่น (Shake Effect) โดยระบบ**ไม่หักเวลาสะสม**
* ในกรณีที่ไม่ต้องการเล่นต่อในด่านนี้ คุณสามารถกดปุ่ม **"ข้ามด่าน" (Skip)** เพื่อเล่นด่านถัดไปได้ทันที

### 3. การจำกัดเวลาและการนับคะแนน (Timer & Score Tracking)
* **การจับเวลา:** มีเวลาจำกัด **3 นาที (180 วินาที)** ต่อด่าน
* **คะแนน:** เมื่อเจอจุดต่าง คะแนนจะเพิ่มขึ้นทีละ 1 คะแนน ระบบจะรวมคะแนนและเวลาที่เหลือสะสมจากทุกด่านไปโชว์สรุปผลเมื่อจบด่านสุดท้ายบนหน้าจอจบเกม (End Screen)

## 🛠️ โหมดนักพัฒนา (Developer Mode Details)

เครื่องมือช่วยสำหรับนักพัฒนาเพื่อระบุและปรับปรุงพิกัดจุดต่างบนแผนที่รูปภาพ:

### 1. การเปิดใช้งานโหมดนักพัฒนา
สามารถเข้าใช้งานได้ 2 วิธี:
* กดปุ่มทางแป้นพิมพ์อักษร **`D`** (หรือ **`d`**) เพื่อสลับเปิด/ปิดแผงควบคุมนักพัฒนา (Developer Panel)
* เปิดผ่าน URL query parameter โดยพิมพ์ต่อท้ายลิงก์: `index.html?dev=true`

### 2. การสร้างและทดสอบพิกัด (Coordinates Logging & Copy JSON)
* เมื่อเปิดโหมดนักพัฒนาแล้ว ให้คลิกบนภาพฝั่งขวา (ภาพสำหรับจับผิด)
* ทุกการคลิกจะเป็นการวางวงกลมจำลองแสดงขอบเขตของการตรวจจับพิกัด
* พิกัดเปอร์เซ็นต์แกน `X`, `Y` และรัศมีเริ่มต้น `R` จะถูกบันทึกเป็นรูปแบบ JSON ในกล่อง Textarea ทันที
* นักพัฒนาสามารถคัดลอก (Copy) ข้อมูลพิกัด JSON เหล่านี้ไปแปะในตัวแปร `levels` ในไฟล์ [game.js](file:///Users/mac/spot-differences/game.js) เพื่ออัปเดตตำแหน่งจุดต่างอย่างถาวรได้
* เมื่อเปลี่ยนใจหรือต้องการทดสอบจุดใหม่ สามารถกดปุ่ม **"ล้างข้อมูล" (Clear)** เพื่อเริ่มจับพิกัดใหม่ได้ทันที (โดยการทำความสะอาดจะล้างเฉพาะวงกลมของนักพัฒนาที่สร้างขึ้นใหม่เท่านั้น ไม่กระทบกับวงกลมคะแนนปกติของเกม)
